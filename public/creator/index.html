I had no real option. No tool that was truly local, truly unlimited, truly fast. So instead of compromising — I built one myself.

> FILE://AEROGRAB.LOG SIGNATURE BUILD

AeroGrab — the world's first one-handed, gesture-driven file transfer baked directly into a file manager — was designed, architected, and coded end-to-end by Afsar.

One closed fist on the sender's camera lifts a file into the air. One open palm on the receiver's camera pulls it down. No tap. No share menu. No upload. The file moves directly device-to-device over WebRTC, while a real on-device hand AI (MediaPipe Tasks Vision) reads the gesture in the browser — nothing leaves your network.

Behind the magic: a custom Socket.IO signaling layer, a WebRTC P2P bridge, an LAN auto-discovery protocol (Hevi Network), and a strict gesture state machine that rejects accidental hands. Built from scratch as part of the Hevi Explorer ecosystem — by Afsar Ali, for TWH.
